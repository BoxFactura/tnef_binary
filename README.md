# Tnef Binary Gem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tnef_binary_gem', git: 'git://github.com/arandilopez/tnef_binary_gem.git'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install tnef_binary_gem
